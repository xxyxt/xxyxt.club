Eclipse Tomcate创建Web项目

Posted on 2019-05-28 | In 学习笔记 | | View
Words count in article: 253 | Reading time ≈ 1

前期准备：

  • Tomcate下载：https://tomcat.apache.org/
    下载免安装版，下载完成直接解压到jdk的安装目录
  • 在eclipse中配置Tomcate：可以参考这篇文章
    https://www.cnblogs.com/Leo_wl/p/4769760.html

遇到的问题

  • Q1、配置完Tomcate之后，在启动过程中点击Window->showview->Servers，找不到server选项。

  • 解决办法：参考下面的连接
    https://blog.csdn.net/wssmiss/article/details/88069484

    • Q2、通过Tomcate的根目录中的bin目录下的starup.bat可以启动Tomcate服务，打开 http://localhost:8080 中也可以看到Tomcate的介绍页面，但是新建的web工程启动服务之后一直404。
      通过starup.bat可以启动成功

      在eclipse中新建web工程启动服务是 一直404

  • 解决办法：参考链接
    https://blog.csdn.net/x123453316/article/details/80951239
    很简单的解决办法：就是.jsp文件不可以放在WEB-INF文件夹下，要在Webcontent目录下新建文件夹，将.jsp文件放入新建的文件夹里面就可以了。

    运行成功
